TokenLens — Privacy Policy

Last updated: May 8, 2026

TokenLens collects zero data. Everything runs locally in your browser.

What TokenLens Does

TokenLens is a Chrome extension that counts tokens in real-time on AI chat platforms (ChatGPT, Claude, and Gemini). It helps you track how much of the AI model's context window you've used.

Data Collection

We collect absolutely no data. Specifically:

  • No personal information is collected
  • No conversation content is stored, transmitted, or logged
  • No analytics or tracking scripts are used
  • No cookies are set
  • No network requests are made by the extension — enforced via connect-src 'none' in the Content Security Policy

How Token Counting Works

All token counting happens 100% locally in your browser using a JavaScript-based tokenizer. The extension reads the visible text on the page to count tokens. This text is:

  • Processed entirely in-memory within your browser tab
  • Never sent to any server, API, or third party
  • Never stored permanently — only held in memory while the tab is open

Local Storage

The extension uses chrome.storage.local to save only:

  • Widget position (x/y coordinates) — so the floating widget stays where you put it
  • Overlay toggle state — whether the widget is visible or hidden
  • Minimized state — whether the widget is expanded or collapsed

All per-tab token data is automatically deleted when the tab is closed.

Permissions

TokenLens requests only two Chrome permissions:

  • activeTab — To read text on the current AI platform tab for token counting
  • storage — To save widget position and display preferences locally

No other permissions are requested. Specifically, there is no scripting, tabs, history, or webRequest permission.

Third-Party Services

TokenLens uses zero third-party services. There are no external scripts, no CDNs, no remote resources of any kind loaded by the extension.

Open Source

The extension's source code is fully inspectable. You can verify every claim in this privacy policy by reading the code yourself.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected in a new version of the extension. The "Last updated" date at the top will be changed accordingly.

Contact

If you have any questions about this privacy policy or the extension, please open an issue on the project's repository or contact the developer directly.
